How to use the 4-print_square module
===================================

The module contains the `print_square(size):` that prints square of size
of param size.

Usage
=====

The print_square receives just one param, `size` and prints a square of
height and width of size `size`

::
    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(1)
    #

    >>> print_square(5)
    #####
    #####
    #####
    #####
    #####


If the size is zero, then nothing is printed

::
    >>> print_square(0)



Bad usage
=========

size must be of type int

floats
::
    >>> print_square(3.5)
    Traceback (most recent call last):
    TypeError: size must be an integer

string
::
    >>> print_square("string")
    Traceback (most recent call last):
    TypeError: size must be an integer

list
::
    >>> print_square([])

size must be >= 0

::
    >>> print_square(-7)
    Traceback (most recent call last):

typecheck first before value check
